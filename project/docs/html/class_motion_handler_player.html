Detailed Description

The handler for the player's actuators, which in this case are the two wheel actuators. Its main job is to translate the directional commands from the user into the appropriate differential drive wheel speeds.

MotionHandlerPlayer manages the modification to the velocity based on user input and collisions. PlayerMotionBehavior translates velocity and position to a new position. Both of these are straightforward, but the framework allows for more sophisticated models of motion in which each wheel has distinct speed.

For this iteration, both wheels are always going at maximum speed, and cannot be controlled independently.
